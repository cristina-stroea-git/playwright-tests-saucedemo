Saucedemo Playwright Tests
Automated end-to-end tests for the https://www.saucedemo.com/ web application using Playwright.

Project Overview
This project covers:
- Login functionality (data-driven tests for multiple users)
- Shopping cart operations
- Checkout process (Step 1, Step 2, Complete)
- Validation of required fields and error messages
- 
How to Run
1. Clone the repository:
git clone https://github.com/<your-username>/playwright-tests-saucedemo.git
cd playwright-tests-saucedemo

3. Install dependencies:
npm install

4. Run tests:
npx playwright test

Notes
Tests use the Page Object Model (POM) for maintainability.
Data-driven tests allow easy addition of multiple users or checkout scenarios.
